# AI应用开发进阶学习路线
> **按阶段 / 模块化学习，每个阶段形成完整能力闭环。**

这样你能：

* 理解为什么学这一块
* 知道学到什么程度算够
* 知道如何落地项目
* 更容易和招聘要求对齐

---

我下面直接给你整理成一份：

> **面向前端转 AI 工程 / AI IDE / AI 应用开发 的完整阶段式细化学习清单（求职导向版）**

结构统一按：

1. **学习目标（为什么学）**
2. **核心知识点（学什么）**
3. **推荐资料 / 开源项目 / 官方文档**
4. **实践任务 / 项目要求（做什么）**
5. **阶段产出标准（学到什么程度算完成）**
6. **对应招聘/面试重点（为什么企业要你学这个）**

# 第一阶段：LLM 基础认知 + Prompt 工程 + API 应用开发

---

# 学习目标

建立：

> **AI 产品开发的基础认知和最底层能力。**

让你理解：

* 大模型如何工作
* 为什么 Prompt 有效
* 如何正确调用模型 API
* 如何开发最基础的 AI 产品

这是所有 AI 岗位默认要求。

---

---

# 需要学习的内容

---

## 1. LLM 基础原理（理解级）

### 学习内容：

* Token 是什么
* Embedding 是什么
* Attention 是什么
* Transformer 为什么有上下文能力
* Temperature / Top_p / Max Token 的作用
* Context Window 是什么

---

### 为什么学：

因为面试一定问：

> “你理解 LLM 的基本原理吗？”

如果只会调 API 不懂这些：

很容易被判定：

> 只是会调包。

---

---

## 2. Prompt Engineering

### 学习内容：

* Zero-shot Prompt
* Few-shot Prompt
* Chain of Thought
* Role Prompt
* Constraint Prompt
* Prompt Template

---

### 为什么学：

因为：

> Prompt 是所有 AI 产品逻辑控制的基础。

---

---

## 3. 模型 API 接入

### 学习内容：

* Chat Completion API
* Streaming 输出
* System Prompt
* Message History 管理
* 上下文拼接策略

---

### 为什么学：

因为：

> 所有 AI 应用第一步都是接模型。
